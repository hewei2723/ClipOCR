# ClipOCR

ClipOCR 是一款方便快捷的图像文字识别工具，旨在简化用户截图并提取文字的过程。通过运行生成好的 `ClipOCR.exe` 可执行文件，您可以轻松地进行文字提取、处理和其他功能。

## 主要功能

1. **自动文字提取：**
   - 启动软件后，按下截屏键即可自动识别剪贴板中的图像，提取其中的文字。

2. **文字处理模式：**
   - **粘贴模式：** 将提取的文字写入剪贴板，方便用户直接粘贴。
   - **搜题模式：** 将文字发送到对应的API并展示答案，通过通知在Windows 11中显示返回的结果。
   - **文字转语音模式：** 将文字转为语音并将文字保存为txt文件，语音以相应格式保存在同一目录下。

## 如何使用

1. **下载软件：**
   - 尚未发布。请关注 [Release 页面](https://github.com/hewei2723/ClipOCR/releases) 获取最新版本。

2. **启动软件：**
   - 双击运行 `ClipOCR.exe`。

3. **截图并提取文字：**
   - 按下截屏键，软件将自动识别剪贴板中的图像，提取文字。

4. **选择文字处理模式：**
   - 根据需求选择不同的文字处理模式，如粘贴模式、搜题模式、文字转语音模式等。

5. **粘贴或获取结果：**
   - 在粘贴模式下，直接粘贴获取到的文字。在其他模式下，软件将展示处理结果或通知。
